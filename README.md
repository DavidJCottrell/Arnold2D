# Setup

## Dependencies

### Build Tools:

`sudo apt install build-essential cmake`

### ZeroMQ:

`sudo apt-get install libczmq-dev`

### SDL2

`sudo apt-get install libsdl2-dev libsdl2-mixer-dev libsdl2-image-dev`

## Build and Run

`mkdir build && cd build`

`cmake .. && cmake --build . && ./arnold`
